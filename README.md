# MLFF National Dashboard System

A full-stack analytics dashboard platform built using React.js, Node.js, Express.js, PostgreSQL, and JWT Authentication.

---

# Features

## Authentication
- User Registration
- Secure Login
- JWT Authentication
- Password Hashing

## Dashboard Analytics
- Transaction Statistics
- Revenue Analytics
- Pie Charts
- Dashboard Cards
- Transaction Tables

## Backend Features
- REST APIs
- PostgreSQL Integration
- Protected Routes
- Middleware Authentication

## Frontend Features
- Responsive UI
- Modern Dashboard
- API Integration
- Analytics Visualization

---

# Tech Stack

## Frontend
- React.js
- Tailwind CSS
- Axios
- Recharts

## Backend
- Node.js
- Express.js
- PostgreSQL
- JWT
- bcryptjs

# API Endpoints

## Register
POST

```bash
/api/mlff-national/auth/register
```

## Login
POST

```bash
/api/mlff-national/auth/login
```

## Add Transaction Stats
POST

```bash
/api/mlff-national/txn-stats
```

## Get Dashboard Stats
GET

```bash
/api/mlff-national/dashboard-stats
```
